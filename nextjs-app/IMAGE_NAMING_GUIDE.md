# IMAGE NAMING GUIDE FOR FLASHCARDS

## Directory Structure
```
public/images/
├── numbers/
├── colors/
├── family/
├── days/
├── fruits/
├── extended-family/
└── months/
```

## Naming Convention
Use exactly these filenames (case-sensitive):

### Numbers (public/images/numbers/)
- `0.jpg` or `0.png` - for "sufuri" (zero)
- `1.jpg` or `1.png` - for "moja" (one)
- `2.jpg` or `2.png` - for "mbili" (two)
- `3.jpg` or `3.png` - for "tatu" (three)

## Image Requirements
- Format: JPG or PNG
- Size: Recommended 200x200px minimum
- Quality: High resolution for clarity
- Content: Child-friendly, clear, and representative of the word

## After Adding Images
Once you've added your images with these exact names, the flashcards will automatically load them!
